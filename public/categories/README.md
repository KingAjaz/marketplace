# Category Images

Place category images in this directory:

- `foodstuffs.jpg` - Image for Foodstuffs category (fresh produce and grains)
- `meat-protein.jpg` - Image for Meat & Protein category (fresh meat and protein sources)
- `live-animals.jpg` - Image for Live Animals category (live animals for sale)
- `processed-foods.jpg` - Image for Processed Foods category (processed and packaged foods)

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 800x600px or 1200x900px
- **Aspect Ratio**: 4:3 or 16:9
- **File Size**: Optimize to under 500KB for better performance

## Image Suggestions

You can use:
- Stock photos from Unsplash, Pexels, or similar
- Custom photography
- Professional product images

The images will automatically display on the home page categories section. If an image is missing, a colored placeholder with the first letter of the category will be shown.
